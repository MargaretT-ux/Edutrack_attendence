># EduTrack — Classroom Attendance Manager 📋

A lightweight React Native mobile app that helps teachers in under-resourced schools track student attendance quickly, view daily summaries, and monitor attendance history — all without needing internet access.

Built by [Thato Lepelesana](https://github.com/MargaretT-ux) — EdTech developer from Lesotho 🇱🇸, building tools for African educators.

---

## 🎯 Problem This Solves

In many schools across Lesotho and the broader African continent, teachers still track attendance on paper registers. This leads to:
- Lost or damaged records
- No quick way to see trends or patterns
- Difficulty identifying students who are frequently absent

EduTrack replaces the paper register with a simple, fast mobile tool that works offline.

---

## 📱 Features

- ✅ **Mark attendance** — Present, Late, or Absent for each student
- 📊 **Daily summary** — Instant count of present, absent, and late students
- 📅 **Attendance history** — View and edit records for any past date
- 👤 **Manage students** — Add or remove students from the class list
- 🎨 **Clean UI** — Designed for quick use in a busy classroom
- 📶 **Offline first** — No internet connection required

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- Expo CLI installed (`npm install -g expo-cli`)
- Expo Go app on your phone (iOS or Android)

### Installation

```bash
# Clone the repository
git clone https://github.com/MargaretT-ux/edutrack-attendance.git

# Navigate into the project
cd edutrack-attendance

# Install dependencies
npm install

# Start the development server
npx expo start
```

Scan the QR code with **Expo Go** on your phone to run the app.
